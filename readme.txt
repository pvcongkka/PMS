PAC55xx Example Code v1.4.0

This release adds support for the PAC5527 device.
For more detailed version history, see .\pac55xx_peripheral_examples\project_version.h

Make sure to download the appropriate PAC55xx support package for the desired IDE
using a myActive login from the following link:
https://active-semi.com/products/intelligent-motor-control/pac5523#et_pb_tab_4


IAR, Keil, and Eclipse IDE projects are provided in the example code.

The pac55xx_simple folder contains a simple GPIO toggle example.  This is the easiest 
code base to get started on PAC55xx firmware development. 

The pac55xx_peripheral_example contains example code for the majority of the PAC55xx
device peripherals.
